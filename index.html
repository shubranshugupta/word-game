WORD GAME

Game Rules

  1. Only guess 5 letter Words.
  2. Guess the right word in 6 attempts.
  3. If you guess correct word then all box will be green.
  4. If you guess correct letter in same position then box will be green.
  5. If you guess correct letter in different position then box will be gold.
  6. If box blink red then the word is invalid.
